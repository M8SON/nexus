# Nexus

Local-first shared assistant framework for Claude and Codex — a nexus point for AI agents working under `/home/daedalus/linux`.

`nexus` replaces the older `claude-recall` standalone tool by combining session recall, local-doc recall, and shared agent policies under one repo. It's also designed to host an active-memory layer (via MemPalace) with per-repo wing scoping.

The retired `claude-recall` source has been archived to `~/.archive/claude-recall-2026-04-26/` rather than deleted, preserving its git history for reference.
